### Button

The `Button` component provides a customizable button interface with support for different styles and configurations.

#### Props

The `Button` component accepts props defined by `ButtonProps<T>`, extended with additional configurations:

| Name                 | Type                                | Default        | Description                                                                                                                                                                                                              |
| -------------------- | ----------------------------------- | -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `href`               | `string`                            |                | URL for anchor tag behavior (if rendered as a link).                                                                                                                                                                     |
| `color`              | `keyof ButtonColors`                | `"light"`      | Color variant of the button (`dark`, `failure`, `gray`, `info`, `light`, `purple`, `success`, `warning`, `blue`, `cyan`, `dark`, `gray`, `green`, `indigo`, `light`, `lime`, `pink`, `purple`, `red`, `teal`, `yellow`). |
| `fullSized`          | `boolean`                           | `false`        | Whether the button should span full width.                                                                                                                                                                               |
| `gradientDuoTone`    | `keyof ButtonGradientDuoToneColors` |                | Dual-tone gradient color variant for the button.                                                                                                                                                                         |
| `gradientMonochrome` | `keyof ButtonGradientColors`        |                | Monochrome gradient color variant for the button.                                                                                                                                                                        |
| `target`             | `string`                            |                | Target attribute for anchor tag behavior (if rendered as a link).                                                                                                                                                        |
| `isProcessing`       | `boolean`                           | `false`        | Indicates if the button is in a processing/loading state.                                                                                                                                                                |
| `processingLabel`    | `string`                            | `"Loading..."` | Label displayed during the processing/loading state.                                                                                                                                                                     |
| `processingSpinner`  | `ReactNode`                         |                | Custom spinner element displayed during the processing/loading state.                                                                                                                                                    |
| `label`              | `ReactNode`                         |                | Label or content displayed inside the button.                                                                                                                                                                            |
| `outline`            | `boolean`                           | `false`        | Whether the button should have an outlined style.                                                                                                                                                                        |
| `pill`               | `boolean`                           | `false`        | Whether the button should have rounded corners (pill style).                                                                                                                                                             |
| `size`               | `keyof ButtonSizes`                 | `"md"`         | Size variant of the button (`xs`, `sm`, `md`, `lg`, `xl`).                                                                                                                                                               |
| `icon`               | `boolean`                           | `false`        | Indicates if the button should display an icon.                                                                                                                                                                          |
| `...props`           | `ButtonProps<T>`                    |                | Additional props passed down to the underlying `ButtonBase` component.                                                                                                                                                   |
