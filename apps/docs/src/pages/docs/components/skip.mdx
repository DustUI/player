### Skip Control

The `SkipButton` component is a React component that provides skip control functionality integrated with Video.js for video playback.

#### Props

The component extends `ButtonProps` from the `Button` component and includes an additional `seek` prop for specifying the skip duration.

| Name   | Type                                | Description                              |
| ------ | ----------------------------------- | ---------------------------------------- |
| `seek` | `5 \| 10 \| 30 \| -5 \| -10 \| -30` | For this number of seconds have an icons |

#### Example Usage

```jsx copy
import { SkipButton, MediaPlayer } from "@dust/player";

function InitializePlayer() {
  return (
    <MediaPlayer
      options={{
        autoplay: false,
        controls: true,
        responsive: true,
        aspectRatio: "16:9",
        sources: [
          {
            src: "https://example.com/video.mp4",
            type: "video/mp4",
          },
        ],
      }}
      className="w-full"
    />
  );
}

function VideoPlayerControls() {
  return (
    <div className="video-controls">
      <SkipButton color="blue" size="md" icon seek={10} />
    </div>
  );
}

function App() {
  return (
    <>
      <InitializePlayer />
      <VideoPlayerControls />
    </>
  );
}
```

### Functionality

- **Skip Control**: Clicking the button skips the video playback forward or backward by the specified `seek` duration in seconds.
- **Video Player Integration**: Utilizes `videoPlayer.seek()` method to skip video playback.

### Integration Notes

- Ensure that `videoPlayer` is correctly initialized and accessible within the scope where `SkipButton` is used.
- Customize the appearance and behavior of the button using props such as `color`, `size`, and `icon`.
